## 🚀 Roadmap & Next Steps

### 1. New Features & Ideas
- **Minigames**: Integrate a simple embedded game (e.g., HTML5 Canvas or React) to make the app more engaging for the family.
- **Transferable Assets**: Allow users to fully transfer ownership of an asset (like a loyalty card) to another user, rather than just sharing access to it.
- **AI Integration**:
  - Connect to an AI service (e.g., Gemini / OpenAI) to auto-suggest tasks or categorize events.
  - Create a special "AI Group" where you can add tasks, and the AI will process or auto-complete them.
- **Group Chat**: Implement a real-time chat window per group using Firestore's real-time listeners.
- **Task Assignment Constraints**: Ensure tasks/events in the Personal Calendar cannot suggest assigning to other users. Assignment suggestions should only happen when sharing on a group calendar.
- **Autosave Engine**: Implement autosaving when editing events/tasks so progress isn't lost.

### 2. Backlog
- **Push Notifications**: Cloud functions to ping family members when new events/tasks are created.
- **Android Compilation**: Wrap the web/PWA into a native Android APK build once the web version is feature-complete.

## 🛠 Handover Notes
If development is interrupted (e.g., PC shutdown), refer back to this document to see the exact state of the project. Always remember to run:
1. `npm run build`
2. `git add .`, `git commit -m "..."`, `git push`
3. `npx firebase-tools deploy --only hosting` (Requires manual login via browser on new PCs)
